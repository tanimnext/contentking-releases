# Content King — Releases

Official desktop builds of **Content King** (AI Content Generator).

⬇️ **[Download the latest version](https://github.com/tanimnext/contentking-releases/releases/latest)**

| Platform | File |
|----------|------|
| macOS    | `ContentKing_Mac_vX.Y.Z.dmg` — open, drag to Applications |
| Windows  | `ContentKing_Windows_vX.Y.Z.zip` — unzip, run `ContentKing.exe` |

The app checks this repository automatically and shows an in-app
**Update Available** banner when a new version is published.

*This repository hosts release builds only.*
